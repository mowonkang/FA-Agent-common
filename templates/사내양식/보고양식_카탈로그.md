## ⚠️ 사용 원칙 (먼저 읽기)

- 본 카탈로그의 양식·톤·색·구분별 사이즈·레이아웃은 **참고 가이드**일 뿐
  강제 규칙이 아니다. **자료 구성·가독성·스토리 향상을 위해 자율 조정**해도 된다.
- 단, **절대 조건**: 글자가 칸에 안 맞거나 화면 밖으로 삐져나오면 안 된다.
  넘치면 보조 슬라이드로 분리하거나 문구를 압축한다.
- **최우선 목표**: 임원 보고에 **바로 쓸 수 있는 퀄리티 + 짜임새 + 깔끔한
  구성 + 스토리**. 양식 모방보다 메시지 전달이 우선.
- 색·폰트·사이즈 기본값은 `LGES_PPT_작업_가이드.md` / `LGES_양식_톤_지침.md`
  참고. 충돌 시 가독성·무오버플로를 우선한다.

## 메인 + 보조 조립 권장

`LGES_PPT_작업_가이드.md` 0-A "메인 1~2 + 보조 N" 구조와 매핑:

- **메인(1~2장)** — 블록 **A** 의 4사분면 요약(현황·문제·방안·결정포인트).
- **보조(N장)** — 주제별로 블록 선택:
  - 수치 진척 → **C** (KPI/투자비 표)
  - 전환 논리 → **G** (As-Is/To-Be)
  - 일정 → **B** (Master Plan)
  - 기술 상세 → **F** (구성·사양)
  - 추진체계 → **D**, 전략·컨셉 → **E**, 다음 단계 → **H**
