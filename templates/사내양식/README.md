### 임원 보고 양식 카탈로그

`(보고양식)YYMMDD_보고제목_v1.0.pptx` 의 12종 슬라이드를 8개 재사용 보고
블록으로 정리한 카탈로그:

→ [**`보고양식_카탈로그.md`**](./보고양식_카탈로그.md) (사람용) /
[`보고양식_카탈로그.json`](./보고양식_카탈로그.json) (ppt-writer 판독용,
`python3 scripts/build_report_catalog.py` 재생성)

양식·톤·사이즈는 **참고**일 뿐 강제 아님 — 가독성·스토리를 위해 자율 조정
가능하나, **텍스트가 박스를 벗어나선 안 되고** 임원 즉시 활용 품질이 최우선.
적용 예시: `scripts/build_fa_exec_report.py` →
`outputs/FA자동화_임원보고_예시_2026-05-18.pptx` (메인 1 + 보조 4, 스토리
6단계, references 출처 인용).

## LGES 양식 톤 지침서

본 폴더 양식들의 공통 디자인 톤(슬라이드 사이즈·폰트·색상·placeholder 패턴 등)을 자동 분석·정리한 지침서:

→ [**`LGES_양식_톤_지침.md`**](./LGES_양식_톤_지침.md)

새 양식·신규 슬라이드 작성 시 본 지침을 따르면 LGES 임원 보고 톤이 유지됩니다.

핵심 요점:
- **슬라이드 사이즈**: 27.52cm × 19.05cm (LGES 그룹 표준, 16:9 아님)
- **폰트**: `LG스마트체 Regular` (한글) + `Arial Narrow` (영문/숫자)
- **색상**: 본문 `#000000` 검정 / 강조 `#006600` 다크 그린 / placeholder `#FFFF00` 노랑
- **폰트 크기**: 본문 12~13pt, 헤더 14pt, 표 셀 10pt, 각주 9pt

## 호출 방법

`ppt-writer` 는 `templates/**/<이름>.pptx` 로 재귀 검색하므로, 양식 이름만 지정해도 됩니다:

- "사내양식 회의록 PPT 작성해줘"
- "회의록 PPT 작성해줘" (다른 폴더에 동명 양식 없으면 자동 매치)

같은 이름의 `.md` 양식이 있거나 동명 `.pptx` 가 여러 폴더에 있으면 ppt-writer 가 어느 경로인지 되묻습니다.

## 자리표시 표시 방식

이 폴더의 양식은 **그대로 둡니다** — `{{ key }}` 토큰을 직접 박지 않아도 됩니다.

ppt-writer 가 다음 단서로 자리표시를 자동 인식합니다:
- 자연어 자리표시 텍스트 (`이름`, `장소`, `20YY년`, `xx:00`, `요약 내용 1`, `ㅇㅇㅇ`, `OOO` 등)
- 노란·초록 등 색상 강조 run

자세한 패턴 목록은 본 폴더의 [`LGES_양식_톤_지침.md`](./LGES_양식_톤_지침.md) "3. placeholder 패턴" 섹션을 참조하세요.
